Nawtpad

A code editor for Windows/.Net.

  • Syntax highlighting.
  • Incremental search.
  • Overview scrollbar.
  • Brace matching.
  • Autoindentation on paste.
  • Multi-window editing.
  • Themes/schemes.
  • Comment hotkey.
  • Bookmarks.

Download (100 KB, .Net 3.5 required, code included)

ngen install Nawtpad.exe to reduce startup time

Known limitations: monospace only, no fancy unicode stuff.
